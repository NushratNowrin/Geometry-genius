Know about DOM basic FAQ

DOM stands for Document Object Model. Everything in an HTML file is in document form. Those documents are converted into object structures in the DOM. Since that object file is not the main file of HTML, but a model, it is called Document Object Model. When a webpage is loaded in the browser, the HTML document is converted into a DOM tree.

The purpose of DOM:

  • DOM can change HTML elemenst.
  • DOM can change HTML element's style.
  • DOM can create new HTML element.
  • DOM can remove any HTML elements.
  • DOM can change HTML element's attribute.
  • DOM can create new HTML event.

There are 5 ways to select elements in a DOM using selectors.

  1. getElementsByTagName().
  2. getElementsByClassName().
  3. getElementById()
  4. querySelector()
  5. querySelectorAll()

Event bubbling is a DOM event propagation where the event first triggers on the child target element, and then triggers on the parent of the target element in the same nesting hierarchy one by one till it reaches the outermost DOM element.

Callback functions are necessary because many JavaScript actions are asynchronous. Instead, it executes in the background while the rest of the code runs. A callback's purpose is to execute code in response to an event.
